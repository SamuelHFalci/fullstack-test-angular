Fullstack Test - Angular + NodeJs

Instructions:

  1. You will work on the pages/recipes component
  2. You can visualize your work on this link
  3. The main goal is to feed a bootstrap table with data comming from the api
  4. Clone the api project with the command git clone https://github.com/SamuelHFalci/fullstack-test-node.git
  5. Install the backend dependencies with yarn or npm i command
  6. Run the backend with the command yarn start or npm run start

    7. Backend job:

  7. Notice that the DataTable perform a get request with query params to the backend api
  8. You have to implement the pagination on the api side using the query params
  9. Hint: There is a class for handling database stuff on the backend api

    10. Frontend job:

  10. After retrieve the paginated data correctly from the backend make it possible to click on the recipe name and open a modal with the recipe Name, url, description, author, ingredients and steps
  11. Hint: You can use the ngx-bootstrap modal component to do this
  12. Make it possible for click on the recipe url and open the url in a new browser tab
  13. The table should look like the example below:
    14. Name Url Description Author
    {{ recipe.Name }} {{ recipe.url }} {{ recipe.Description }} {{ recipe.Author }}
