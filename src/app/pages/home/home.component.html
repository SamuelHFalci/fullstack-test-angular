Fullstack Test - Angular + NodeJs

Instructions:

  1. Clone this repository (git clone https://github.com/SamuelHFalci/fullstack-test-angular.git) and create a branch with your own name. (git checkout -b your-name)
  2. You will work on pages/recipes component
  3. You can visualize your work on this link
  4. The main goal is to feed a bootstrap table with data comming from the api
  5. The api is hosted on the api folder and you can adjust the code for matching the requirements
  6. The data pagination must be implemented on both, frontend and backend side, using page size and fetching just the data it's needed
  7. The table should look like the exemple below.
    8. Name Url Description Author
    {{ recipe.Name }} {{ recipe.url }} {{ recipe.Description }} {{ recipe.Author }}
  8. Notice that the data is hardcoded on the example and the pagination is being done just on the frontend side.
